# Project skills

Agent skills for this repo live here. Anyone who clones the repository gets them automatically in Cursor (project skills).

| Skill | Trigger |
|-------|---------|
| `bingo-generator` | `/bingo-generator`, “make a new bingo sheet” |
| `carnatic-quiz-generator` | `/carnatic-quiz-generator`, “build quiz video”, “make a quiz video” |

**Canonical location:** edit skills under `.cursor/skills/` only. Mirror to Claude Code with:

```bash
./scripts/sync-skills.sh
```
